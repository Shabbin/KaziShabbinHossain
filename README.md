## Origin & Ownership

This project started from a publicly available starter repository shared by a YouTube creator.  
It was used only as an initial scaffold to avoid repetitive setup and boilerplate.

I did **not** follow a step-by-step tutorial or guided instructions.

From the beginning, development was driven by my own design, UX, and implementation decisions, including:

- Custom layout and section structure
- Rebuilt navigation and menu interactions
- Original animation logic and motion timing
- Visual redesign and styling choices
- Incremental refactoring as the project evolved

The current codebase represents my independent work and ongoing iteration built on top of an initial foundation.
